FAQ

1. What is Ging?

Ging is a Live system that you can burn on a CD. It is based on Debian GNU/kFreeBSD (which, as its time, is based on Debian, GNU and the kernel of FreeBSD).

Ging consists entirely of free software (as per Debian Free Software Guidelines) and has a compromise to remain this way.

3. What does "Ging" stand for?

It stands for "Ging Is Not Ging". It is, to my knowledge, the first recursive acronym with an implicit contradiction.

4. Where can I download it?

Currently, Ging is distributed through the eDonkey network:

Download iso image (via ed2k).

Download gpg-signed sha1sum (via http).

5. How to build an image?

Use the scripts from glibc-bsd svn repository (check svn.debian.org), in trunk/web/livecd/, plus the patches in trunk/web/ging/.
