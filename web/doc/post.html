Miscellaneous setup

You can setup your soundcard (and other misc stuff) with the debconf templates in kfreebsd-config package.

To configure your keyboard mapping, install the kbdcontrol package and select your keyboard layout from the list displayed.

If you're setting up a firewall, you can use the OpenBSD Packet Filter. Currently this is hacked in the base system but there's a pf package in preparation.

To satisfy apt-get's gnupg checking requirements, the following command (as root) will do:

gpg --keyserver subkeys.pgp.net --no-default-keyring --primary-keyring /etc/apt/trusted.gpg --recv 19BFB8AB

X.Org

X server configuration is done normaly via debconf (dpkg-reconfigure xserver-xorg).

Everything is as you would find on Debian GNU/Linux, except the mouse setup, whose device names are different. The correspondence between mouse devices and their device node names is:

/dev/ums* USB
/dev/psm* PS/2
/dev/mse* Bus mouse
/dev/cuaa* Serial
/dev/sysmouse mouse daemon (moused)

With regard to mouse daemon, you only need it if you use your mouse in the console (via syscons). Otherwise you don't have to install it. FIXME: explain how to use syscons.

Development

When building packages make sure you have the latest autotools-dev package installed. Many packages have outdated config.* files. See PORTING for more information.
