X.Org

Mouse configuration is now handled via debconf (dpkg-reconfigure xserver-xorg). The correspondence between mouse devices and their device node names is:

/dev/psm0 PS/2
/dev/cuaa* Serial
/dev/ums* USB
/dev/mse0 Bus mouse
/dev/sysmouse mouse daemon (moused)

With regard to mouse daemon, you only need it if you use your mouse in the console (via syscons). Otherwise it can be safely removed. FIXME: explain how to use syscons.

If you're having trouble running X as a non-root user, the following script/hack has been used successfully.

#!/bin/bash -e
export DISPLAY=:0
sudo xinit /usr/bin/sudo -u `whoami` x-window-manager -- -nolistentcp

Development

When building packages make sure you have the latest autotools-dev package installed.
Many packages have outdated config.* files. See PORTING for more information.

Miscellaneous

Stopping and starting daemons is done the Debian way with /etc/init.d/foo start|stop.
