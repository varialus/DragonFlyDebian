2.1. Supported hardware

Debian GNU/kFreeBSD's hardware support is outlined in the 'Ð…upported hardware' section of the FreeBSD handbook. Almost all device drivers included in a 'GENERIC' FreeBSD kernel configuration are available in the respective Debian kernel package(s), with the exception of certain non-free binary-only drivers and GPL-conflicting code. Currently supported hardware platforms include i386 and amd64.

2.2. Install media

Our current installation CD, based on a modified FreeBSD sysinstall, serves as temporary solution until a full port of the debian-installer is complete. Bootable images for both i386 and amd64 can be found at http://glibc-bsd.alioth.debian.org/install-cd. These images usually aren't much more than 100MB in size, making them perfect for lower capacity mini CD/DVDs and flash drives.

debian-installer based images are now also available, both for kfreebsd-amd64 and kfreebsd-i386. Given they are generated daily and based on unstable, they may be broken from time to time. However they should be prefered to the FreeBSD based installer. See the debian-installer manual for the installation procedure.

2.3. Disk space

Disk space requirements for Debian GNU/kFreeBSD don't differ too much from standard Debian GNU/Linux, ranging from less than several hundred megabytes for a fresh base install, to several gigabytes for a complete graphical desktop environment and applications.
