Supported hardware

Debian GNU/kFreeBSD runs on the i386 architecture ("IA32", "x86") and supports any hardware that FreeBSD does. See "Supported hardware" in the FreeBSD handbook for details.

Installation media

We have an installation CD that is similar in functionality to the debian-installer "netinst" flavour. Note, however, that debian-installer is not ported yet, so our install CD is actualy using the FreeBSD installer (if you have installed FreeBSD before, this will look suspiciously familiar).

You can find releases of our install CD here.

For people with network shortage (or aiming at re-distribution), there's a set of CD images containing the complete package archive here. These images are for use with apt-cdrom (or similar) once the system has been installed. They're NOT suitable for installation of a base system.

For information on how to proceed, check "Installing Debian GNU/kFreeBSD" section.

Disk space

If you're planning to use your GNU/kFreeBSD system for development purposes (building packages etc.), around 2Gb of hard disk space is a good idea. If you're just giving Debian GNU/kFreeBSD a try, you could probably get away with around 500mb.
