Supported hardware

Debian GNU/kFreeBSD runs on the i386 architecture ("IA32", "x86") and supports any hardware that FreeBSD does. See "Supported hardware" in the FreeBSD handbook for details.

Installation media

The official installation method is the Live CD. It contains a minimal environment for bootstrapping a Debian GNU/kFreeBSD system. However, one thing the live CD is not currently capable of, is partitioning.
To create the required partitions/slices, you will need a working Linux or BSD system.
You can achieve this with an existing installation or from a FreeBSD install/live CD.

Whatever software you use to do this, the important thing is that it has to support creating a FreeBSD partition and a root and swap slice. You can find more information on partitioning with FreeBSD here.
