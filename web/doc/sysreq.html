Supported hardware

At the time of writing, Debian GNU/kFreeBSD is available for the i386 and amd64 architectures and features hardware support roughly equivalent to FreeBSD/i386 or FreeBSD/amd64, respectively. See "Supported hardware" in the FreeBSD handbook for details.

Installation media

Our custom installation CD is similar in functionality to the debian-installer "netinst" flavour. Be aware that despite this fact, a port of the official debian-installer to GNU/kFreeBSD has not yet been completed. In the mean time our install CD is based on the regular FreeBSD installer (aka sysinstall. If you have installed FreeBSD before, this will look suspiciously familiar).

Releases of our install CD can be found in the install-cd/ directory of this site.

For information on how to install, proceed to "Installing Debian GNU/kFreeBSD".

Disk space

If you're planning to use your GNU/kFreeBSD system for development purposes (building packages etc.), around 2GB of hard disk space is a good idea. If you're just giving Debian GNU/kFreeBSD a try, you could probably get away with around 500MB.
