Pre-installation notes

Important - Make sure you've read "Installation media" before you continue. It contains important information.

Installing Debian GNU/kFreeBSD

Insert the Debian GNU/kFreeBSD installation CD and boot your computer from it.

Once the CD is finished booting, you should be presented with an installation menu.

Select either Custom or Express install. Do NOT select Standard installation because this will leave you with a broken system.

You will now be asked to create your partitions (if you're unsure, a simple / partition and swap is fine).

For more information on partitioning with FreeBSD, see this page of the FreeBSD handbook.

After you've finished creating the necessary partitions, you will be asked to install a boot manager.
GRUB can load kFreeBSD via /boot/loader, so if you're going to be using GRUB as your boot loader,

Select "None - Leave the master boot record untouched".


Otherwise install FreeBSD bootloader to MBR as normal.

When you are at the "Choose distributions" screen, select "Minimal".
It is important that you select the minimal option. This CD only contains a minimal installation, and if you want to install more software you can use the Debian tools later to do so.

The "Choose installation media" section is next.

Select "CD/DVD - Install from a FreeBSD CD/DVD".

The base system will now be installed.

Once the base system is extracted, all the base packages need to be installed and configured (this might take a while on slow machines). Press ALT-F3 to switch to the terminal where they're being setup, and wait for system configuration to finish. You'll have to answer a pair of questions about timezone and debconf setup.

When the process is complete, the FreeBSD installer will ask if you want to do any further setup. Answer no to all of these questions and exit the installation system.

You should now have a functioning Debian GNU/kFreeBSD system. See "Boot loader" for information on how to boot your new system with GRUB.

Boot loader

If you use GRUB as your bootloader, the following configuration can be used to boot Debian GNU/kFreeBSD.

title Debian GNU/kFreeBSD
root (device)
kernel /boot/loader

If you installed your system on /dev/ad0s1a (the systems first partition/slice), (device) would be (hd0,0,a).
